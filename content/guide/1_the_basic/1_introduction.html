# Function Components và Class Components

Đầu tiên, chỉ Class component mới có state nhưng từ khi Hook được giới thiệu thì có thể thêm state cho Function Components (Hook được giới thiệu trong React Native 0.59 (opens new window)). Vì Hooks là tính năng phổ biến trong tương lai để viết React components nên ta sẽ dùng Function Components, thỉnh thoảng khi cần thì dùng Class Components

Xem thêm ví dụ về Class Component ở những docs version cũ (opens new window)
